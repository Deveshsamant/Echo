# Echo AI — Master Test Sheet
A comprehensive list of prompts to test all features of the Echo AI Agent.

## 🎙️ Voice & Conversation Mode
1. **"Echo, what time is it?"** *(Wait for response... DO NOT say Echo again, just say:)* **"And what's the date?"**
2. **"Echo, tell me a joke."** *(Wait...)* **"Tell me another one."** *(Wait...)* **"Stop."** *(Should exit conversation mode).*
3. **"Echo, set my volume to zero."** *(Wait...)* **"Now set it back to 50."**

## 👁️ AI Vision & Camera
4. **"Use your vision to read my screen. What application is open?"**
5. **"Take a look at my screen using vision and tell me if there are any error messages."**
6. **"What do you see through the camera right now?"** *(Make sure camera is running on Dashboard).*
7. **"Take a photo with the webcam."**

## 📂 File Intelligence & Management
8. **"Find the PDF file I was reading yesterday."**
9. **"Search my computer for a file related to 'Echo_Operations'."**
10. **"Create a new text file on my desktop called 'echo_test.txt' and write 'Hello World' inside it."**
11. **"Delete the file 'echo_test.txt' from my desktop."**

## 📅 Calendar & Reminders
12. **"Add a calendar event for tomorrow at 2 PM called 'Team Meeting'."**
13. **"What's on my calendar for tomorrow?"**
14. **"Remind me in 2 minutes to check my email."** *(Should create a calendar event with a reminder time).*

## 🌐 Web & Browser Control
15. **"Open YouTube and search for 'Lofi Girl live stream'."** *(Keep watching to ensure it actually clicks the video to play it).*
16. **"What browser tabs do I have open right now?"**
17. **"Switch to the YouTube tab."**
18. **"Close the current browser tab."**

## 💻 System & Windows Control
19. **"What's my current CPU and RAM usage?"**
20. **"Snap the current window to the right side of the screen."**
21. **"Minimize the current window."**
22. **"Switch to my other virtual desktop."**
23. **"Show a notification that says 'Echo is fully operational!'."**

## 💬 Telegram Integration
24. *(Send from Telegram)* **"Take a screenshot and send it to me."**
25. *(Send from Telegram)* **"What is my PC doing right now?"**

## 🦸‍♂️ Complex / Multi-Step Autonomous Tasks
26. **"Download the latest Python installer from python.org and tell me when it's done."**
27. **"Open Notepad, write a short poem about a robot, and save it as 'robot_poem.txt' in my Documents folder."**
28. **"Search Google for 'today's weather in London', read the result on screen, and tell me what it says."**
